# Agentic Patterns

Reference implementations for bounded agentic workflows.

The examples are deliberately small and non-flashy. The challenge in enabling
agentic workflows, when you know how creative agents can become, is setting and
maintaining boundaries that don't unnecessarily limit autonomy while still
providing a measure of safety.
