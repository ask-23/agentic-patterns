# Agentic Patterns

Reference implementations for bounded agentic workflows.

The examples are deliberately small. The interesting parts are the boundaries: where the workflow can act, where it must evaluate, where it must escalate, and where it must stop.

## Patterns

1. [Research with eval gate](01-research-with-eval-gate/README.md)

## Intentional limits

- No full agent framework.
- No web UI.
- No vendor lock-in.
- No claim of autonomy without containment.

The point is workflow shape.
